## 📖 Overview

**Queen Mencia Web Pair** is a modern, elegant web-based pairing interface designed for WhatsApp bot authentication. Generate your pairing codes effortlessly with a stunning animated UI featuring blue and yellow gradients, 9-color cycling input fields, and smooth glassmorphism effects.

> 👑 *"Connect with elegance. Pair with power."*

## ✨ Key Features

- **🎨 Modern Glassmorphism Design** – Sleek, semi-transparent UI with blur effects
- **🔄 9-Color Cycling Input** – Dynamic gradient animation for the pairing number field
- **🌊 Animated Background** – Seamlessly shifting blue and yellow gradients
- **📱 Fully Responsive** – Works perfectly on desktop, tablet, and mobile devices
- **⚡ Smooth Transitions** – Hover effects, scaling buttons, and elegant animations
- **🔐 WhatsApp Bot Pairing** – Dedicated interface for generating pair codes
- **🎯 One-Click Pairing** – Simple and intuitive user experience
